## 一、选品是闲鱼赚钱的核心

### 选品不是"喜欢什么卖什么"
> 而是 **"什么好卖卖什么"**

### 好品的5个标准

```
□ 1. 有搜索量：闲鱼搜索框搜一下，有竞品在出单
□ 2. 有利润空间：进货价 < 售价的50%（毛利>50%）
□ 3. 竞争适中：同款卖家 < 50个
□ 4. 好发货：轻小（<1kg），不易碎
□ 5. 可复购：用完了还会再买（耗材类最佳）
```

### 选品渠道

| 渠道 | 方法 | 难度 |
|:----:|:----|:----:|
| **闲鱼搜关键词** | 搜"充电器"→看哪些卖得好→找同款货源 | ⭐ |
| **拼多多热销榜** | 看拼多多什么卖得好→搬到闲鱼加价 | ⭐ |
| **淘宝热销榜** | 同上 | ⭐ |
| **1688热搜** | 看什么品搜索量大 | ⭐⭐ |
| **抖音爆款** | 抖音带火的品→闲鱼卖 | ⭐⭐ |
| **小红书种草** | 小红书被种草的品→闲鱼卖 | ⭐⭐ |

## 二、4大货源渠道详解

### 1️⃣ 拼多多（最适合新手）
```
优势：价格最低，一件代发，不需要囤货
流程：
  ① 在拼多多找到想卖的商品
  ② 截图+复制标题
  ③ 挂到闲鱼加价30-50%
  ④ 有人拍下→去拼多多下单→地址填买家
  ⑤ 赚差价

注意：
  - 不要用拼多多原图（会被判断为倒卖）
  - 自己拍照或修图
  - 藏好拼多多水印
```

### 2️⃣ 1688（进阶）
```
优势：批发价更低，可定制
劣势：大部分不一件代发，需囤货

适合：确定爆品后批量进货
```

### 3️⃣ 义乌购
```
优势：小商品天堂，价格极低
适合：家居小物/手机配件/创意礼品
```

### 4️⃣ 虚拟资源（0成本）
```
电子书/教程/模板/PPT/简历
AI生成资料包
整理打包 → 闲鱼卖（1-9.9元）
优势：0成本，100%利润
```

## 四、定价策略

### 闲鱼定价公式
```
售价 = 进货价 × 2（或进货价 + 50%）

但别忘了：
- 闲鱼用户喜欢砍价
- 标价要比心理价高10-20%
- 给买家"赚到了"的感觉
```

### 定价技巧

| 技巧 | 说明 | 例子 |
|:----:|:----|:----:|
| 尾数定价 | 用9结尾 | 39元→39.9元 |
| 锚定定价 | 标原价vs现价 | "原价99现价49" |
| 捆绑销售 | 多件优惠 | "两个包邮75" |
| 阶梯定价 | 新旧不同价 | 全新69/95新49 |

## 五、供应链管理

### 一件代发 SOP
```
买家下单
  ↓
去拼多多下单（地址填买家）
  ↓
把拼多多单号复制到闲鱼发货
  ↓
等买家确认收货
  ↓
去拼多多确认收货（有些自动确认）
  ↓
利润到账
```

### 备货模式 SOP
```
1. 先测试（上架3-5个品看数据）
2. 确定爆品（持续出单的品）
3. 小批量进货（3-5件试水）
4. 销量稳定后批量进（10-50件）
5. 优化包装+物流成本
```
