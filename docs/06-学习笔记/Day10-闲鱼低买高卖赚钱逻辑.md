## 一、低买高卖的底层逻辑

### 一句话总结
> **你在闲鱼上赚的每一分钱，都是信息差的钱**

### 信息差的3个层次

| 层次 | 说明 | 利润空间 |
|:----:|:----|:--------:|
| **低级信息差** | 知道哪里便宜（拼多多→闲鱼） | 30-50% |
| **中级信息差** | 知道什么值钱（别人不认识的宝贝） | 100-500% |
| **高级信息差** | 知道谁在买+谁能卖（对接供需） | 500-1000%+ |

### 老黄要赚哪个层次的钱？

```
低级（拼多多代发）：✅ 新手入门，稳赚不赔
中级（捡漏/识货）：✅ 适合有专业知识的人
高级（供需对接）：⚠️ 需要资源，暂时不做
```

**关键操作要点**：

| 步骤 | 操作 | 注意 |
|:----:|:----|:----:|
| 选品 | 拼多多搜热门商品，找销量高+评价好的 | 避开品牌官方店（控价严） |
| 定价 | 拼多多价×1.3~1.5 | 留出砍价空间（标价再高10%） |
| 图片 | 用拼多多买家秀/自己拍 | **不要直接用商家图**（会被识别倒卖） |
| 描述 | 编个"故事"（搬家/买多了/冲动消费） | 增加真实感 |
| 发货 | 拼多多下单→地址填买家 | 隐藏拼多多面单（撕掉或涂改） |

### 模式2：捡漏→高价卖（利润高，需要眼力）

**货源渠道**：

| 渠道 | 方法 | 利润 |
|:----:|:----|:----:|
| **闲鱼自己** | 搜"搬家"/"分手"/"不懂"等关键词找低价 | 100-500% |
| **转转/拍拍** | 二手平台低价买入 | 50-200% |
| **线下旧货市场** | 古玩城/旧货市场淘货 | 500%+ |
| **品牌清仓** | 618/双11后收转卖 | 30-80% |

**闲鱼捡漏关键词**：

```
"搬家急出" → 卖家急着卖，价格通常很低
"分手了" → 情绪低价，能砍狠
"不懂/不会用" → 卖家不知道价值
"公司发的" → 0成本来的，好砍价
"前男友/前女友送的" → 低价处理
```

**典型捡漏案例**：
```
捡漏：华为手环（卖家"不懂不会用"）→ 50元收
卖出：检查没问题 → 闲鱼卖150元
利润：100元（200%利润率）
操作时间：30分钟
```

### 模式3：漏洞/折扣品转卖（需要信息灵通）

```
1. 关注品牌折扣信息
2. 抢到优惠价商品
3. 转手加价卖掉
4. 赚取差价
```

**常见折扣来源**：
| 渠道 | 折扣力度 | 抢购难度 |
|:----:|:--------:|:--------:|
| 天猫U先 | 1-3折试用装 | ⭐ |
| 拼多多百亿补贴 | 8-9折 | ⭐ |
| 京东秒杀 | 5-7折 | ⭐⭐ |
| 品牌内购群 | 3-5折 | ⭐⭐⭐ |
| 跨境电商（别样/海淘） | 国外打折=国内赚钱 | ⭐⭐⭐ |

**实测利润**：
```
天猫U先9.9元小样 → 闲鱼卖19.9-29.9元
利润：10-20元/单
一天抢10个 → 日入100-200元
```

## 四、低买高卖的核心公式

### 选品公式
```
好品 = (需求大) × (竞争小) × (差价高) × (好发货)

评分标准（每项1-10分）：
需求大：搜索量>1000/天（闲鱼搜一下有大量在卖）
竞争小：同款卖家<50家
差价高：利润率>40%
好发货：重量<1kg，不易碎

总分>30分 → 好品
总分20-30分 → 测试品
总分<20分 → 放弃
```

### 定价公式
```
基础公式：售价 = 进货价 × 1.5~2.0

闲鱼实际定价 = 售价 × 1.1（留出砍价空间）
最终成交价 ≈ 售价 × 0.9（买家一般会砍10%）

保证：利润 ≥ 进货价的50%
```
