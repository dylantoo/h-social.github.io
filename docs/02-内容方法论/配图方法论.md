# 配图方法论

> AI配图生成规范和组合策略

---

## 配图方案

| 图片 | 数量 | 尺寸 | 来源 | 说明 |
|:----:|:----:|:----:|:----:|------|
| 封面 | 1 | 1080×1440 | HTML模板 | 品牌形象 |
| 内容卡 | 1-2 | 1080×1440 | HTML模板 | 信息呈现 |
| AI配图 | 2-3 | 1024×1024 | Kolors | 意境/氛围 |

## 总图组合
> **封面 + 1-2张内容卡 + 2-3张AI配图 = 4-6张**

## AI配图规范（Kolors）

### 模型
- `Kwai-Kolors/Kolors`
- 端点：`api.siliconflow.cn`

### Prompt 模板
```
英文prompt, 暗色水墨风/极简插画风, 意境氛围,
no text no watermark
```

### 风格选择
- 辟谣严肃类 → 暗色/水墨风/极简
- 生活轻松类 → 明亮/插画风/温暖
- 需根据内容主题匹配

### 注意事项
- 英文prompt效果更稳定
- 末尾加 `no text no watermark` 避免生图带文字
- 尺寸 1024×1024（小红书会裁剪，关键内容放中间）

## 水印处理
- **内容**：「反生活」
- **位置**：图片右下角
- **颜色**：白色半透明（需根据背景调整）
- **工具**：PIL/Pillow Python脚本
