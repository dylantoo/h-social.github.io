# 📕 《爆款小红书》学习笔记

> 核心知识点：从0到1做小红书的完整方法论
> 适用：老黄「反生活」账号实战参考

---

## 一、账号定位四大维度

### 1. 赛道选择
- **宽赛道**：流量大但竞争激烈（美妆、穿搭）
- **窄赛道**：流量小但精准，容易做垂直IP（生活辟谣 ✅）
- **结论**：「反生活」选的是窄而精的赛道，方向正确

### 2. 人设打造
- **专业度**：可信的来源和数据
- **差异点**：毒舌风格区别于严肃科普
- **记忆点**：「反生活」名字本身就有记忆点

### 3. 封面统一性
- 统一色调、字体、排版
- 让用户刷到就知道是你的内容
- ✅ 我们已经有了HTML模板封面体系

### 4. 内容垂直度
- 小红书算法会打标签，垂直度越高推荐越精准
- ❌ 不要发无关内容（哪怕是想蹭热点）

## 二、爆款公式

```
爆款 = 60%选题 × 20%标题 × 10%封面 × 10%运气
```

### 选题三原则
1. **覆盖足够广**：这个话题有多少人关心？
2. **情绪价值**：能让人"啊对对对"或"什么？原来是这样"
3. **收藏价值**：用户看完会不会收藏？

### 标题公式
- **痛点前置**：「这5个习惯正在毁掉你的...」
- **颠覆认知**：「你以为的养生其实是减寿」
- **数字量化**：「90%的人都做错了...」
- **制造悬念**：「医生不会告诉你的...」

## 三、小红书流量密码

### 推荐算法优先级
```
收藏 > 评论 > 转发 > 点赞 > 完播/完读
```

**关键洞察**：收藏是小书最看重的指标，代表"有用"
→ 每篇文章末尾加"自查清单/避坑指南"提高收藏率

### 流量池层级
| 层级 | 曝光量 | 判断标准 |
|:----:|:------:|---------|
| 初级 | 200-500 | 点击率 > 10% |
| 中级 | 2K-5K | 互动率 > 5% |
| 高级 | 1W-5W | 收藏率 > 8% |
| 热门 | 10W+ | 综合评分 |

### 最佳发布时间
- 早：7:00-9:00（通勤）
- 午：11:30-13:30（午休）
- 晚：17:00-19:00（下班）
- 夜：21:00-23:00（睡前）
- ⚠️ 但不要固定时间发（反检测需要）

## 四、起号黄金7天

### Day 1-3：养号期
- 每天刷30分钟，搜索竞品，点赞收藏评论
- 不要急着发内容

### Day 4-7：密集发布期
- 每天1-2篇，快速测试选题方向
- 数据好的继续深挖，数据差的果断放弃

### 冷启动提醒
> "前100个付费用户要手搓" — OPC极简创业原则
> 「反生活」现在就是在手搓阶段，先手工找公式再自动化
