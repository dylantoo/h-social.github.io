# 📕 《疯传》学习笔记

> Jonah Berger 著 — 让内容像病毒一样传播的6大原则
> 适用：让「反生活」的内容更容易被转发和传播

---

## 核心框架：STEPPS 六原则

```
S — Social Currency（社交货币）
T — Triggers（触发器）
E — Emotion（情绪）
P — Public（公共性）
P — Practical Value（实用价值）
S — Stories（故事）
```

---

## 一、Social Currency 社交货币

### 核心概念
人们分享内容是为了**提升自己在别人眼中的形象**。

### 应用
- 分享辟谣内容 = "我是个有见识、不容易被骗的人"
- 转发「反生活」= "我帮朋友避坑，我很有用"

### 实操方法
| 方法 | 举例 |
|------|------|
| 制造内幕感 | "医生不会告诉你的真相" |
| 稀缺性 | "这篇内容仅存24小时" |
| 彰显品味 | "聪明人都知道的5个生活常识" |

---

## 二、Triggers 触发器

### 核心概念
**经常想到的东西，更容易被分享。** 把内容和日常高频场景绑定。

### 应用
- **绑定场景**：「反生活」内容应该和用户日常动作挂钩
  - 喝水时 → "你喝的水真的干净吗？"
  - 吃饭时 → "这种食物致癌？真相是..."
  - 刷手机时 → "睡前刷手机真的会失明？"
- **绑定节日**：节气内容是天然触发器（立夏到了→该讲什么辟谣）

---

## 三、Emotion 情绪驱动

### 核心概念
**高唤醒情绪**的内容更容易被分享。

| 高唤醒（促进传播） | 低唤醒（抑制传播） |
|:---:|:---:|
| 愤怒 😤 | 满足 😌 |
| 敬畏 😮 | 悲伤 😢 |
| 兴奋 🎉 | 倦怠 😴 |
| 幽默 😂 |  |

### 「反生活」的应用
- **愤怒**："这种伪科学害了多少人！" ✅
- **幽默**：毒舌吐槽本身就带着幽默感 ✅
- **敬畏**："科学实验证明..." ✅
- **避免**：纯说教（低唤醒，没人转发）

---

## 四、Public 公共性

### 核心概念
**看到别人在做，自己也会想尝试。** 让内容"可见"。

### 应用
- 引导评论区互动 → 内容有热度信号
- 鼓励@好友 → "@你那个正在XXX的朋友"
- 水印「反生活」→ 截图传播时自带品牌 ✅

---

## 五、Practical Value 实用价值

### 核心概念
**有用的信息传播最快。** 人们喜欢帮别人省时省钱省力。

### 实操
- **清单体**：「5个自查清单」→ 收藏率极高
- **省钱角度**："帮你省了XX元" ✅
- **避坑角度**："90%的人踩过的坑"

### 「反生活」天然优势
辟谣内容的本质就是 = **帮别人省钱+避坑** ✅✅

---

## 六、Stories 故事包装

### 核心概念
**信息在故事中传播最快。** 不要干讲道理，要讲故事。

### 实操方法
| 方法 | 举例 |
|------|------|
| 个人经历 | "我朋友上周..." |
| 用户故事 | "粉丝投稿说..." |
| 案例故事 | "XX市的大爷..." |

### 毒舌辟谣的故事公式
```
某人因为信了谣言 → 发生了什么后果
→ 我查了资料发现真相 → 毒舌吐槽 → 帮大家避坑
```

---

## 对「反生活」的启发总结

| STEPPS原则 | 当前做法 | 可以改进 |
|------------|---------|---------|
| Social Currency | ✅ 辟谣=聪明人选择 | 增加"内幕感"角度 |
| Triggers | ✅ 节气绑定 | 增加更多生活场景绑定 |
| Emotion | ✅ 愤怒+幽默 | 增加"敬畏"（科学数据） |
| Public | ✅ 水印品牌 | 鼓励@好友转发 |
| Practical Value | ✅ 省钱避坑 | 多用清单体提升收藏 |
| Stories | ⚠️ 有时缺故事 | 每篇开头加个人经历/案例 |
