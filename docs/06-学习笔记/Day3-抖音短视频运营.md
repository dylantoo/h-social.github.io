# 📕 Day3: 抖音短视频运营

> **核心：抖音的算法机制和变现逻辑**
> 来源：行业报告 + 头部博主经验

---

## 一、抖音vs小红书

| 维度 | 小红书 | 抖音 |
|:----:|--------|------|
| 内容形式 | 图文为主 | 短视频为主 |
| 用户群 | 一二线女性多 | 全年龄段 |
| 内容生命周期 | 3-7天 | 几小时-1天 |
| 完播率要求 | 中等 | 极高（>30%） |
| 变现门槛 | 1000粉 | 0粉可带货 |
| 变现天花板 | 中等 | 极高 |

---

## 二、抖音算法：完播率是王

### 推荐机制
```
发布 → 200-500初始流量
  ↓ 完播率>30% → 点赞率>3%
2000-5000流量池
  ↓ 完播率>40% → 互动率>5%
1万-10万流量池
  ↓ 综合评分
继续扩大/停止推荐
```

### 核心指标排序
```
完播率 > 复播率 > 点赞率 > 评论率 > 转发率 > 关注率
```

---

## 三、抖音短视频爆款公式

### 前3秒定生死
```
前3秒：钩子（制造好奇/焦虑/冲突）
   ↓
3-15秒：内容主体（信息密度要高）
   ↓
15-30秒：高潮（反转/数据/结论）
   ↓
最后5秒：行动号召（关注/评论/转发）
```

### 标题和封面
- 大字封面 + 强对比色
- 标题：痛点/好奇/争议
- 视频封面和文字要一致

## 四、抖音变现模式

### 1. 短视频带货
| 方式 | 说明 | 门槛 |
|:----:|------|:----:|
| 商品橱窗 | 视频挂车 | 0粉（2025新规） |
| 精选联盟 | 选品带货 | 1000粉 |
| 自营商品 | 自己的产品 | 不限 |

### 2. 直播带货
- 抖音最大的变现来源
- 日播4-6小时
- 客单价50-200元最佳

### 3. 星图广告
- 类似小红书蒲公英
- 门槛：1万粉
- 报价：1万粉≈500-2000元

### 4. 知识付费
- 抖音学浪/小课
- 短视频引流→直播间转化
- 适合技能/教育类

---

## 五、反生活×抖音

### 内容适配
```
小红书图文
  ↓ 改编
抖音短视频（30-60秒）
  → 前3秒钩子
  → 快速打脸3个谣言
  → 毒舌收尾
```

### 起步建议
```
1. 先做好小红书，别分心
2. 等小红书稳定出爆款（5000粉+）再开抖音
3. 内容一鱼多吃：小红书图文→抖音口播
4. 抖音主要用于引流到微信私域
```
