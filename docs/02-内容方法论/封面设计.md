# 封面设计

> HTML模板封面设计规范

---

## 核心规范

| 参数 | 值 | 说明 |
|------|:---:|------|
| 尺寸 | 1080×1440px | 小红书3:4竖屏 |
| 格式 | JPEG | 质量 90%+ |
| 水印 | 「反生活」 | 右下角统一位置 |
| 风格 | 深色现代感 | 渐变/纯色背景 |

## 设计方案

### 方案A：纯CSS卡片（推荐，最稳定）
- 纯CSS布局，不依赖外部图片
- 渐变背景 + 大标题文字 + emoji装饰
- 颜色：深紫/深蓝暗色系为主，标题用亮色
- 文字加 text-shadow 保证可读性

### 方案B：AI图背景
- Kolors生图作为背景 + 半透明遮罩 + 文字叠加
- 需要把AI图base64嵌入HTML（文件自包含）
- 适合意境类内容

## 颜色方案

| 用途 | 色值 | 场景 |
|------|:----:|------|
| 标题红 | #FF6B6B | 错误/怼人 |
| 辟谣绿 | #43A047 | 正确做法 |
| 毒舌金 | #FF8F00 | 吐槽高亮 |
| 背景深色 | #1a1a2e | 暗色背景 |
| 卡片强调 | #FF6B6B | 左边框6px |

## 截图流程

```bash
# Chrome headless 截图
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless --screenshot=output_raw.png \
  --window-size=1080,1440 \
  --force-device-scale-factor=2 \
  "file:///path/to/cover.html"

# 缩放+转JPEG
sips -z 1440 1080 output_raw.png --out cover.jpg -s format jpeg -s formatOptions 90
```
