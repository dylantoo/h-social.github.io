# 文案公式

> 「反生活」毒舌辟谣赛道的文案结构和写法

---

## 标准结构

```
┌─────────────────────────┐
│   1. 钩子开头（前30字）    │ ← 制造反差/好奇/对立
├─────────────────────────┤
│   2. 抛谣言（50-80字）    │ ← "你是不是也听过..."
├─────────────────────────┤
│   3. 逐条辟谣（300-400字）│ ← 3-5条，每条"错法→正法"
├─────────────────────────┤
│   4. 自查/总结（50-80字）  │ ← 帮助读者对号入座
├─────────────────────────┤
│   5. 毒蛇收尾（30字）      │ ← 犀利吐槽，引发互动
├─────────────────────────┤
│   6. 正文500-600字         │
└─────────────────────────┘
```

## 详细拆解

### 1. 钩子开头（前30字）
- **制造对立**："你妈是不是也说过，XXX？"
- **制造好奇**："XX亿人都在做的事，竟然是错的？"
- **制造焦虑**："别再XXX了，你的XX正在被毁掉"

### 2. 抛谣言
- 先说出常见的错误认知
- 用"你是不是/是不是也/我经常看到"拉近距离
- 增加真实感："我闺蜜上个月..."

### 3. 逐条辟谣（核心部分）

每条结构：
```
❌ 错法：原来XXX的说法
→ ✅ 正法：实际是XXX
💥 数据/来源：根据XXX研究/WHO指南
```

**3-5条**，每条用emoji标注：
- 🥶 常见误解1
- 🩸 常见误解2  
- ☠️ 常见误解3（最离谱的放中间）
- ⚠️ 特别注意

### 4. 自查/总结
- 帮读者判断自己有没有踩坑
- 以清单/表格形式呈现
- 提高收藏率（"先码住"）

### 5. 毒舌收尾
- 吐槽："听君一席话，白读十年书"
- 反差："花钱买智商税，不如请我喝奶茶"
- 互动引导："你们还听过什么离谱的谣言？评论区展开说说"

## 标签规则

- 7-10个标签
- 必含：`#反生活`
- 包含：`#辟谣 #生活科普 #冷知识 #智商税` 等
- 2-3个话题标签
- 1-2个热门标签（蹭流量）

## 示例

> **嘴唇发紫就是心脏不好？省省吧，你嘴唇只是冻着了**
>
> 你是不是也听过"嘴唇发紫=心脏有问题，赶紧去查"这种话？
> 我姥姥就是这么信的，每年冬天都要拉着我去看心内科😅
>
> ❌ 嘴唇发紫=心脏病前兆
> ✅ 嘴唇发紫的学名叫"发绀"，可以是很多原因：
> 🥶 冻的（最常见！天冷嘴唇血管收缩）
> 🩸 缺氧（高原/剧烈运动）
> 😷 贫血（需要查血常规）
> 🔥 中毒（极少数，但需要就医）
>
> 真正需要警惕的是：**长期不明原因的发绀+胸闷气短**
> 别一看嘴唇发紫就往心脏上想，先问问自己：是不是没穿秋裤？💥
>
> 💡自查清单：
> □ 天冷时发紫？→ 正常现象，多喝热水
> □ 运动后发紫？→ 休息一下
> □ 长期+胸闷？→ 去医院
>
> 你妈还说过什么让你紧张的"伪常识"？评论区说说看👇
>

## 三套模板

为避免被判定AI生成，随机切换：

| 模板 | 开头风格 | 语气 |
|:----:|---------|------|
| 故事型 | "我朋友/妈/对象..." | 唠嗑感 |
| 数据型 | "研究发现..." | 偏理性 |
| 吐槽型 | "我真是服了，还在XXX？" | 毒舌max |
