## 分析维度

### 1. 账号定位
| 维度 | 描述 |
|------|------|
| 账号名称 | |
| 粉丝量级 | |
| 内容赛道 | |
| 人设风格 | （严肃/幽默/专业/毒舌） |
| 更新频率 | |
| 变现方式 | |

### 2. 内容分析
| 维度 | 分析要点 |
|------|---------|
| 爆款选题 | 哪些话题数据好？为什么？ |
| 封面风格 | 文字大小、配色、布局 |
| 文案结构 | 开头方式、信息密度、互动引导 |
| 配图质量 | 生图/实拍/设计稿 |
| 标签策略 | 哪些标签流量大 |

### 3. 互动分析
- 评论区的核心讨论方向
- 高赞评论的共性
- 互动引导方式（提问/投票/征集）

### 4. 成长路径
- 从多少粉开始爆发
- 爆款的共性特征
- 阶段性的内容策略变化

## 学习-改进闭环

```
每日浏览竞品（5-10分钟）
  ↓
记录可复用的方法
  ↓
应用到自己的下一篇内容
  ↓
对比数据验证效果
  ↓
持续迭代
```

## 「反生活」的差异化机会

| 竞品特征 | 反生活的应对 |
|---------|-------------|
| 多数辟谣号严肃科普 | 毒舌吐槽，增加传播性 |
| 纯文字/长图文 | 精美HTML模板+AI配图 |
| 追热点为主 | 节气+热点双线驱动 |
| 无标准化流程 | CDP全自动发布，规模化 |
