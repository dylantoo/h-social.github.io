## 一、核心原则：滑梯效应

### 什么是滑梯效应？
> 文案的第一句话就要让读者**滑向**第二句话，第二句滑向第三句... 停不下来。

### 实现方法
1. **第一句话必须超级短**（不超过10个字）
2. **创造好奇心缺口** — 让读者想知道后面是什么
3. **每句话都引向下一句**，没有断点

### 反生活应用
```
❌ 错误开头：
"关于嘴唇发紫和心脏病之间的关系，很多人存在误解..."

✅ 正确开头：
"你嘴唇发紫？先别慌。"
→ 为什么？因为你大概率只是冻着了...
```

## 三、标题写作公式

### 4U原则
```
Urgent（紧迫） + Unique（独特） + Ultra-specific（极度具体） + Useful（有用）
```

### 8种经典标题类型

| 类型 | 公式 | 反生活例子 |
|:----:|------|-----------|
| 1. 数字型 | "N个XX，最后一个99%人不知道" | "5个你以为健康的习惯" |
| 2. 反差型 | "XX不是XX，而是XX" | "嘴唇发紫不是心脏病，是冻着了" |
| 3. 疑问型 | "为什么XX却XX？" | "为什么每天跑步反而伤膝盖？" |
| 4. 否定型 | "别再XX了" | "别再喝这3种'养生茶'了" |
| 5. 指南型 | "XX的正确方式" | "泡脚的正确方式，你一直做错了" |
| 6. 数据型 | "研究显示XX%" | "研究显示：90%的XX都是错的" |
| 7. 故事型 | "我XX后发现XX" | "我姥姥信了30年的养生法" |
| 8. 对比型 | "XX vs XX，差距这么大" | "医院vs民间偏方，差距有多大" |

## 四、辟谣文案的滑梯结构

```
【钩子】第一句话（10字以内，制造好奇）
   ↓
【场景】你是不是也...（带入感）
   ↓
【痛点】这3个谣言坑了多少人（逐条打脸）
   ↓
【反转】真相是...（数据/来源）
   ↓
【高潮】最离谱的是...（最狠的放中间）
   ↓
【升华】自查清单（收藏价值）
   ↓
【收尾】毒舌吐槽 + 互动引导
```

## 五、情绪控制节奏

### 文案的情绪曲线

```
强度↑
    |  ╱ 反转（震惊）
    | ╱  痛感（共鸣）
    |╱   钩子（好奇）
    └────────────→ 时间
        自查（释放）
         毒舌（高潮）
          互动（延续）
```

### 「反生活」的节奏控制
- 开头：节奏快，3-5句话进入正题
- 中间：每条误区独立成段，用emoji区分
- 自查表：节奏放缓，让读者"消化"
- 毒舌收尾：节奏再拉高，留下记忆点

## 六、信任建立技巧

辟谣内容最关键的：**让读者相信你**

| 方法 | 操作 | 重要性 |
|:----:|------|:------:|
| 数据引用 | "WHO/CDC/研究论文" | ⭐⭐⭐ |
| 个人经历 | "我家人就遇到过" | ⭐⭐⭐ |
| 权威背书 | "协和医生朋友说" | ⭐⭐ |
| 逻辑自洽 | "为什么？因为..." | ⭐⭐⭐ |
| 态度诚实 | "这个目前没有定论" | ⭐⭐ |

### ⚠️ 辟谣的底线
> 有数据就列数据，有研究就列研究
> 不确定的就说"目前没有定论"
> 别为了流量制造新的谣言

## 七、行动号召（CTA）

每篇文案末尾必须有明确的引导：

```
💬 评论引导："你听过什么离谱的谣言？评论区说说"
❤️ 收藏引导："先收藏，下次遇到转发给家人"
🔔 关注引导："关注我，每天帮你避坑省钱"
@  艾特引导："@你身边那个最爱转发谣言的亲戚"
```

## 对「反生活」的改进点

| 书中方法 | 当前状态 | 改进方向 |
|---------|:--------:|---------|
| 滑梯效应 | ⚠️ 有时开头不够短 | 前10字必是钩子 |
| 4U标题 | ✅ 基本达标 | 增加"紧迫感"元素 |
| 情绪曲线 | ✅ 结构正确 | 增加自查表阶段的"释放感" |
| 信任背书 | ⚠️ 偶尔缺数据 | 每篇至少1个数据来源 |
| CTA引导 | ✅ 有互动 | 增加@好友引导 |
